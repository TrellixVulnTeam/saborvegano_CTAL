Utilizamos las cookies para los siguientes fines:

(A) Autentificación: utilizamos cookies para identificarle cuando visita nuestro sitio web y mientras navega por él];

(B) Estado: utilizamos las cookies para ayudarnos a determinar si está conectado a nuestro sitio web.

(C) Personalización - utilizamos cookies para almacenar información sobre sus preferencias y para personalizar el sitio web para usted.

(D) Seguridad - utilizamos cookies como elemento de las medidas de seguridad utilizadas para proteger nuestro sitio web y nuestros servicios en general.

(E) Publicidad - utilizamos cookies para ayudarnos a mostrar anuncios que sean relevantes para usted.

(F) Análisis: utilizamos cookies para ayudarnos a analizar el uso y el rendimiento de nuestro sitio web y nuestros servicios.
